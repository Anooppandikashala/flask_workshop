## Session 2 — HTML Fundamentals

### What is HTML?
HTML (HyperText Markup Language) uses **tags** to describe content.  
Think of tags like function calls in C — they wrap content and give it meaning.

```html
<tagname> content </tagname>
```

### Boilerplate — Every HTML File Starts Here

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My First Page</title>
  </head>
  <body>
    <!-- Everything visible goes here -->
  </body>
</html>
```

> **Analogy:** `<head>` is like variable declarations at the top of a C file. `<body>` is your `main()` function.

### Core Tags

#### Text Tags
```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<h3>Smaller Heading</h3>
<p>This is a paragraph of text.</p>
<strong>Bold text</strong>
<em>Italic text</em>
```

#### Links & Images
```html
<a href="https://example.com">Click here</a>
<img src="logo.png" alt="Logo description" />
```

#### Lists
```html
<!-- Unordered list (bullet points) -->
<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

<!-- Ordered list (numbered) -->
<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>
```

#### Layout Tags (Containers)
```html
<div>A generic block container</div>
<span>An inline container</span>
<header>Top section of a page</header>
<nav>Navigation links</nav>
<main>Main content area</main>
<section>A section of content</section>
<footer>Bottom section of a page</footer>
```

#### Forms (important for e-commerce later)
```html
<form action="/submit" method="POST">
  <label for="name">Your Name:</label>
  <input type="text" id="name" name="name" placeholder="John Doe" />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />

  <label for="password">Password:</label>
  <input type="password" id="password" name="password" />

  <button type="submit">Submit</button>
</form>
```

#### Tables
```html
<table>
  <thead>
    <tr>
      <th>Product</th>
      <th>Price</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>T-Shirt</td>
      <td>Rs. 20</td>
    </tr>
    <tr>
      <td>Jeans</td>
      <td>Rs. 50</td>
    </tr>
  </tbody>
</table>
```

## Session 3 — CSS Fundamentals

### What is CSS?
CSS (Cascading Style Sheets) controls how HTML looks.  
> **Analogy:** HTML is the skeleton, CSS is the skin and clothes.

### Three Ways to Add CSS

```html
<!-- 1. Inline (avoid this — hard to maintain) -->
<p style="color: red;">Red text</p>

<!-- 2. Internal (inside <head>) -->
<style>
  p { color: blue; }
</style>

<!-- 3. External (best practice) — link a separate .css file -->
<link rel="stylesheet" href="style.css" />
```

### Selectors

```css
/* Target by tag */
p {
  color: gray;
}

/* Target by class (reusable) */
.card {
  background: white;
  padding: 16px;
}

/* Target by ID (unique, use sparingly) */
#main-title {
  font-size: 32px;
}
```

### The Box Model
Every element is a box:

```
+---------------------------+
|         margin            |
|  +---------------------+  |
|  |       border        |  |
|  |  +--------------+   |  |
|  |  |   padding    |   |  |
|  |  |  +--------+  |   |  |
|  |  |  | content|  |   |  |
|  |  |  +--------+  |   |  |
|  |  +--------------+   |  |
|  +---------------------+  |
+---------------------------+
```

```css
.box {
  width: 300px;
  height: 200px;
  padding: 16px;       /* space inside */
  margin: 24px;        /* space outside */
  border: 1px solid #ccc;
}
```

### Common Properties

```css
.example {
  /* Text */
  color: #333;
  font-size: 16px;
  font-weight: bold;
  text-align: center;
  font-family: Arial, sans-serif;

  /* Background */
  background-color: #f9f9f9;

  /* Spacing */
  padding: 12px 24px;   /* top-bottom left-right */
  margin: 0 auto;       /* center horizontally */

  /* Border */
  border: 2px solid #e2e2e2;
  border-radius: 8px;   /* rounded corners */

  /* Size */
  width: 100%;
  max-width: 600px;
}
```

### Flexbox — The Easiest Layout Tool

```css
.container {
  display: flex;
  justify-content: space-between;  /* horizontal alignment */
  align-items: center;             /* vertical alignment */
  gap: 16px;
}
```

```html
<div class="container">
  <div class="card">Product 1</div>
  <div class="card">Product 2</div>
  <div class="card">Product 3</div>
</div>
```

## Session 4 — JavaScript Fundamentals

### What is JavaScript?
JS makes pages interactive.  
> **Analogy:** Like C's `main()` loop — JS listens for events and responds.

### Adding JS to HTML

```html
<!-- At the bottom of <body> (best practice) -->
<script src="app.js"></script>
```

### Variables

```javascript
// let — can change later (like a regular variable in C)
let count = 0;

// const — cannot change (like a constant in C)
const productName = "T-Shirt";

// var — old way, avoid it
```

### Data Types

```javascript
let name = "Alice";         // string
let price = 29.99;          // number
let inStock = true;         // boolean
let items = [1, 2, 3];      // array (like C array)
let product = {             // object (like C struct)
  name: "Shoes",
  price: 49.99
};
```

### Functions

```javascript
// Function declaration (like a function in C)
function greet(name) {
  return "Hello, " + name + "!";
}

// Arrow function (modern shorthand)
const add = (a, b) => a + b;

console.log(greet("Alice"));   // Hello, Alice!
console.log(add(3, 4));        // 7
```

### DOM Manipulation — Changing the Page

```javascript
// Select an element
const title = document.getElementById("main-title");
const buttons = document.querySelectorAll(".btn");

// Change content
title.textContent = "New Title";

// Change style
title.style.color = "red";

// Add/remove CSS classes
title.classList.add("highlight");
title.classList.remove("highlight");
```

### Events — Reacting to User Actions

```javascript
const button = document.getElementById("cart-btn");

button.addEventListener("click", function () {
  alert("Added to cart!");
});
```

### Exercise 3 — Interactive Product Page
Add to your product page:
- A quantity selector (`+` and `−` buttons)
- JS that updates a displayed quantity number
- A "Add to Cart" button that shows an alert with the product name and quantity


Add this HTML inside the `.product-info` section of `product.html`, replacing the old `<button>`:

```html
<!-- Quantity selector -->
<div style="display: flex; align-items: center; gap: 12px; margin-bottom: 20px;">
  <button id="minus-btn">−</button>
  <span id="quantity-display">1</span>
  <button id="plus-btn">+</button>
</div>

<button id="cart-btn" class="btn-cart">Add to Cart</button>
```

Create `app.js` in the same folder:

```javascript
// The product name we want to show in the alert
const productName = "Classic T-Shirt";

// Track the current quantity
let quantity = 1;

// Get references to the HTML elements we need to control
const display   = document.getElementById("quantity-display");
const plusBtn   = document.getElementById("plus-btn");
const minusBtn  = document.getElementById("minus-btn");
const cartBtn   = document.getElementById("cart-btn");

// Clicking + increases quantity
plusBtn.addEventListener("click", function () {
  quantity++;
  display.textContent = quantity;
});

// Clicking − decreases quantity, but never goes below 1
minusBtn.addEventListener("click", function () {
  if (quantity > 1) {
    quantity--;
    display.textContent = quantity;
  }
});

// Clicking "Add to Cart" shows a confirmation alert
cartBtn.addEventListener("click", function () {
  alert("Added to cart: " + productName + " × " + quantity);
});
```

Link `app.js` at the bottom of `product.html` before `</body>`:

```html
  <script src="app.js"></script>
</body>
```


- `document.getElementById(...)` used correctly to select elements
- `addEventListener("click", ...)` used on all three buttons
- `display.textContent = quantity` updates the number on screen
- The minus button has a guard (`if (quantity > 1)`) so it doesn't go to 0
- The alert includes both the product name and the current quantity

## Session 5 — Bootstrap Introduction

### What is Bootstrap?
Bootstrap is a CSS framework — pre-written CSS classes you can use directly.  
Instead of writing all your own CSS, you use classes like `btn`, `card`, `container`.

> **Why teach it?** It speeds up building UI and looks professional out of the box.

### Adding Bootstrap (CDN — no install needed)

```html
<head>
  <!-- Bootstrap CSS -->
  <link
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
    rel="stylesheet"
  />
</head>
<body>
  ...
  <!-- Bootstrap JS (for dropdowns, modals etc.) -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
```

### Bootstrap Grid System
Bootstrap divides the page into **12 columns**.

```html
<div class="container">
  <div class="row">
    <div class="col-md-4">Column 1</div>
    <div class="col-md-4">Column 2</div>
    <div class="col-md-4">Column 3</div>
  </div>
</div>
```

### Bootstrap Components

#### Navbar
```html
<nav class="navbar navbar-expand-lg bg-dark navbar-dark">
  <div class="container">
    <a class="navbar-brand" href="#">MyShop</a>
    <div class="navbar-nav ms-auto">
      <a class="nav-link" href="#">Home</a>
      <a class="nav-link" href="#">Products</a>
      <a class="nav-link" href="#">Cart</a>
    </div>
  </div>
</nav>
```

#### Card
```html
<div class="card" style="width: 18rem;">
  <img src="https://placehold.co/286x180" class="card-img-top" alt="Product" />
  <div class="card-body">
    <h5 class="card-title">T-Shirt</h5>
    <p class="card-text">A comfortable everyday t-shirt.</p>
    <p class="fw-bold text-success">Rs. 20.00</p>
    <button class="btn btn-primary w-100">Add to Cart</button>
  </div>
</div>
```

#### Button Variants
```html
<button class="btn btn-primary">Primary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-outline-dark">Outline</button>
```

#### Badge & Alert
```html
<span class="badge bg-success">In Stock</span>
<div class="alert alert-warning">Limited stock available!</div>
```

### Key Takeaways

**1 & 2 — Sold Out badge and disabled button**

Replace the Jeans card body with:
```html
<div class="card-body d-flex flex-column">
  <h5 class="card-title">Slim Jeans</h5>
  <p class="card-text text-muted flex-grow-1">Classic slim-fit denim jeans, great for any occasion.</p>
  <p class="fw-bold text-success fs-5">Rs. 50.00</p>
  <!-- Badge shows stock status -->
  <span class="badge bg-danger mb-2" style="width: fit-content;">Sold Out</span>
  <!-- Button is disabled so it cannot be clicked -->
  <button class="btn btn-secondary" disabled>Add to Cart</button>
</div>
```

Key points:
- `badge bg-danger` makes a red badge
- `disabled` attribute on the `<button>` greys it out and stops click events
- A disabled button does NOT need a JS event listener — the browser blocks clicks automatically

**3 — Newsletter subscription form**

Add this section between the products grid and the footer:

```html
<!-- Newsletter Section -->
<div class="bg-light py-5 mt-5">
  <div class="container text-center">
    <h3>Subscribe to Our Newsletter</h3>
    <p class="text-muted">Get the latest deals delivered straight to your inbox.</p>
    <form class="d-flex justify-content-center gap-2 mt-3" onsubmit="handleSubscribe(event)">
      <input
        type="email"
        class="form-control"
        style="max-width: 320px;"
        placeholder="Enter your email address"
        id="newsletter-email"
        required
      />
      <button type="submit" class="btn btn-primary">Subscribe</button>
    </form>
    <p id="subscribe-msg" class="text-success mt-3" style="display: none;">
      Thank you for subscribing!
    </p>
  </div>
</div>
```

JS to add inside the `<script>` block:
```javascript
function handleSubscribe(event) {
  // Prevent the page from reloading (default form behaviour)
  event.preventDefault();

  const email = document.getElementById("newsletter-email").value;
  const msg   = document.getElementById("subscribe-msg");

  // Show the thank-you message and clear the input
  msg.textContent = "Thank you! " + email + " has been subscribed.";
  msg.style.display = "block";
  document.getElementById("newsletter-email").value = "";
}
```

Key points:
- `type="email"` makes the browser validate the email format before submitting
- `required` prevents submitting an empty field
- `event.preventDefault()` stops the browser from refreshing the page on submit
- `style.display = "block"` shows the hidden thank-you message
